# FotoRamkaStik

Описание интерфейса
Структура приложения:
Левая панель - Настройки с вкладками:

Основные: Размеры шаблона и вывода, соотношение сторон

Стикеры: Директория со стикерами, размеры, плотность, ширина полосы

Внешний вид: Настройки перекрытия, поворота, прозрачности, формата вывода

Правая панель - Предпросмотр и управление:

Предпросмотр: Отображение сгенерированной фоторамки

Информационная панель: Текущее разрешение, количество стикеров, статус

Панель управления: Кнопки генерации, сохранения, случайных настроек

Основные функции:
1. Выбор стикеров
Кнопка "Выбрать..." для выбора директории с PNG/JPG/WebP изображениями

Автоматический подсчет количества доступных стикеров

2. Настройка параметров
Размеры шаблона (100-5000px)

Размеры вывода (HD, FHD и другие)

Соотношение сторон (16:9, 4:3, 1:1, 9:16, свободное)

Размеры стикеров (мин/макс)

Плотность заполнения (1-100%)

Ширина полосы для стикеров

Опции перекрытия, поворота, прозрачности

3. Генерация и предпросмотр
Кнопка "Сгенерировать" для создания фоторамки

Реальное время отображения в окне предпросмотра

Автомасштабирование для отображения

Информация о статусе генерации

4. Сохранение результата
Сохранение в PNG, JPEG или WebP формате

Выбор места сохранения через диалоговое окно

5. Дополнительные функции
Кнопка "Случайные настройки" для быстрой генерации с случайными параметрами

Строка состояния с информацией о процессе

Создание тестовых стикеров при первом запуске

Особенности реализации:
Многопоточность: Генерация выполняется в отдельном потоке, не блокируя интерфейс

Адаптивный интерфейс: Разделитель позволяет изменять размер панелей

Валидация: Проверка корректности настроек перед генерацией

Кэширование: Загрузка стикеров выполняется однократно

Обработка ошибок: Подробные сообщения об ошибках в диалоговых окнах

Запуск приложения:
bash
python sticker_frame_gui.py
При первом запуске будет предложено создать тестовые стикеры. После этого можно:

Выбрать директорию со стикерами (или использовать тестовые)

Настроить параметры фоторамки

Нажать "Сгенерировать" для предпросмотра

Сохранить результат в нужном формате

Приложение предоставляет интуитивно понятный интерфейс для создания уникальных фоторамок из стикеров с полным контролем над всеми параметрами генерации.
