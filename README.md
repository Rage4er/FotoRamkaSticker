# Генератор фоторамок из стикеров

## Установка
1. Установите Python 3.8+
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python sticker_frame_gui.py`

## Сборка в EXE
1. Установите PyInstaller: `pip install pyinstaller`
2. Запустите: `build_exe.bat`
3. Готовый EXE файл будет в папке `dist/`

## Использование
1. Выберите папку со стикерами (PNG, JPG, WebP)
2. Настройте параметры во вкладках
3. Нажмите "Сгенерировать" для предпросмотра
4. Сохраните результат
